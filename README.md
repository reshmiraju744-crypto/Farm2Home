# Farm2Home

## Overview

Farm2Home is a Django-based e-commerce web application developed to connect farmers directly with customers, enabling them to buy and sell fresh agricultural products online. The platform provides a secure and user-friendly shopping experience with customer email verification, product browsing, shopping cart, wishlist, product reviews and ratings, and order management.

The application also includes a dedicated farmer dashboard where farmers can manage their products, receive notifications for new customer orders, and monitor their monthly earnings through an interactive chart. The project demonstrates practical knowledge of full-stack web development using Python, Django, responsive web design, authentication, and database management.

## Features

### Customer Features
- Customer registration with email verification
- Secure login and authentication
- Browse and search agricultural products
- Shopping cart management
- Wishlist functionality
- Product reviews and ratings
- Place and manage orders
- View order history
- User profile management

### Farmer Features
- Farmer registration and secure authentication
- Add, edit, and delete products
- Manage product inventory
- Receive dashboard notifications for new customer orders
- View and manage customer orders
- Track monthly earnings through an interactive dashboard chart

### Admin Features
- Manage customers and farmers
- Manage products and categories
- Monitor and manage customer orders
- Manage overall platform activities

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/reshmiraju744-crypto/Farm2Home.git
```

### 2. Navigate to the project directory

```bash
cd Farm2Home
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Apply database migrations

```bash
python manage.py migrate
```

### 7. Run the development server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
